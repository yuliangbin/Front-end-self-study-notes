1、电脑扫盲(DOS命令窗口中的常用的DOS命令)
	- ipconfig / ipconfig -all : windows IP 设置，查看自己电脑上的IP地址
	- ping www.baidu.com -t : 查看自己的网络状况，360的测网速实现原理就是这个。
	- 常用命令
		- ctrl + c : 终止正在运行的DOS命令
		- exit : 关掉DOS命令窗口
		- cls : 清屏
	- 文件夹
		- cd 表示进入当前目录
		- (cd ..)/(cd..):返回当前文件夹的上一级目录
		- cd ./:当前目录
		- cd /:返回当前磁盘的根目录
		- D: 直接进入到某个文件夹/目录
		- cd (文件名/文件夹名): 进入相应的文件目录
		- dir 查看当前文件夹下的目录详细信息
	以上所有操作基本上都是为了进入到具体的某个文件夹下操作，但是我们可以不必这么麻烦，只需要直接找到对应的文件夹，在文件夹的空白处按 shift + 鼠标右键 => 在此处打开命令窗口
