1、如何让JS代码在NODE中执行
	(1)利用webStorm嵌入NODE环境，直接把编写的JS代码通过右键"Run xx.js"在NODE环境中执行即可。(不建议经常这样操作，因为有些时候我最新执行的操作其实会运行前几次的缓存)
	(2)在当前要执行的JS文件下，shift+右键 => 在此处打开命令窗口 => node xx.js .相当于把JS代码在NODE环境里执行了
	(3)在NODE中执行JS代码和浏览器中执行一样，如果JS代码发生改变，想要看到最新的效果需要重新执行对应的JS文件
	(4)在命令窗口中执行　node　,然后开始编写对应的JS代码测试即可(RPEL命令操作)。和在浏览器的控制台中写JS代码是一样的
