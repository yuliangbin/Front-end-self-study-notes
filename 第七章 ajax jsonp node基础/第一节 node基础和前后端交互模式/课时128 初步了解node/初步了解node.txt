================================= NODE 基础知识 =================================
1、	JS是一门运行在"客户端"的"轻量级"脚本编程语言.
	JS不仅可以运行在浏览器端，还可以运行在NODE中。所以NODE和浏览器一样，是一个可以运行JS代码的环境。
	
2、所以什么是NODE？
	NODE是运行JS代码的一个运行环境。我们可以把它等价于浏览器，只不过我们一般会把NODE这个环境安装在服务器端。这样的话我们就可以在服务器端使用JS编写程序，也就是说JS不仅仅是客户端的语言也是服务器端的语言
3、NODE && 浏览器
	- NODE　采用的是谷歌的V8引擎来渲染JS的(运行速度快，稳定，在NODE里编写的代码是不需要考虑兼容问题的) 
	- 浏览器中的全局对象是window，而NODE环境下的全局对象是Global
	- 浏览器是安装在客户端上的，为了保护客户端的安全，基本上不肯能提供用JS对客户电脑磁盘上的文件进行操作的功能(IO操作)。但是在NODE环境下提供了对应的IO操作(服务器上文件的操作)，我们可以使用JS对服务器磁盘下的文件进行增删改查
	- NODE给JS提供了很多新的方法：http.createServer,fs.writeFileSync,fs.readFileSync,...
	
4、什么是NODE？总结：NODE是渲染JS的一个环境，是用V8引擎来渲染JS的。它提供了丰富的IO操作。NODE.JS是基于事件驱动的异步编程(我们在NODE环境下编写的JS程序都是异步编程程序)
